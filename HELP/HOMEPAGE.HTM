OS/tools Products Overview

OS/tools has two basic product lines. Both product lines depend on advanced features of our COMi serial device driver.

COMscope is a designed to help in troubleshooting communications sessions and debugging communications applications.

COMspool provides virtually unlimited serial printer connections for any OS/2 machine.

QuickPage is a single user, single access, program for alphanumeric and digital personal pagers using TAP.

PageStar provides a multiple user, multiple access, alphanumeric and digital pagers using TAP.
